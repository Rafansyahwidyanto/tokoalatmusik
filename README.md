# MusicalInstrumentStore
This program is a web-based CRUD application.
Steps to run the `tokoalatmusik` project using XAMPP:
1. Install XAMPP and start Apache & MySQL from the Control Panel.
2. Copy the `tokoalatmusik` folder to `C:\xampp\htdocs\tokoalatmusik`.
3. Open `http://localhost/phpmyadmin` and create a new database (e.g., `tokoalatmusik`).
4. Import the `tokoalatmusik.sql` file into that database.
5. Open CMD and navigate to the project folder: cd C:\xampp\htdocs\tokoalatmusik
6. Run the built-in CodeIgniter server: php spark serve
7. Open your browser and go to `http://localhost:8080`.
